Documentation for self-serve tools

Services available to all project committers

  • Create a Jira account
  • Create a Git repository: you do this on GitBox, and the documentation is there.
  • Create a mailing list
  • Manage mailing lists: Those with permissions can manage mailing lists via the WebMod service, which lets you view and moderate any pending messages for lists you or your projects manage, and configure moderators for individual lists.
  • Archive a Confluence Wiki space

Services available to all committers of any ASF project

  • Edit your ASF identity: you do this via the Whimsy service, and the documentation is there.
  • Reset your password: you do this via the Apache Account Utility, and the documentation is there.
  • OTP calculator

Services available to all contributors and users

  • Request a Jira account
