Remove a User from a DockerHub Group

Use this form to remove an existing DockerHub user from a team/group in the apache DockerHub organisation. This does not remove the user from the org itself. This action is restricted to infrastructure staff.

The DockerHub username of the person to remove from the group. This must be their DockerHub handle, not their Apache ID or email address.
The name of the existing DockerHub group to remove this user from. For example: my-project
