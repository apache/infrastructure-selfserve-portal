Create a DockerHub Repository

Use this form to create a new repository in the apache DockerHub organisation and grant an existing team write access to it. This action is restricted to infrastructure staff.

The name of the new repository. Must be lowercase and may contain letters, digits, hyphens, underscores, or periods. For example: my-project
A brief, one-line description of the repository (optional).
A longer description of the repository, supports Markdown (optional).
The name of an existing DockerHub team in the Apache org to grant write access to this repository. For example: my-project
