Add a User to a DockerHub Group

Use this form to add an existing DockerHub user to a team/group in the apache DockerHub organisation. The user must already be a member of the Apache org — if not, use the invite user to org form first. This action is restricted to infrastructure staff.

The DockerHub username of the person to add to the group. This must be their DockerHub handle, not their Apache ID or email address.
The name of the existing DockerHub group to add this user to. For example: my-project
